# @fetchmux/core

Shared request, response, and route-receipt contracts (Zod schemas and TypeScript types) for
[FetchMux](https://fetchmux.com), the self-hosted retrieval router for AI agents. Used by
[`@fetchmux/sdk`](https://www.npmjs.com/package/@fetchmux/sdk) and
[`@fetchmux/mcp`](https://www.npmjs.com/package/@fetchmux/mcp); not intended for direct use.

Gateway source and documentation:
[github.com/krutftw/fetchmux](https://github.com/krutftw/fetchmux).

Apache-2.0.
